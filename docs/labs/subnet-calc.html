💡 Cách dùng

Thử nhập các ví dụ trong lý thuyết: 184.156.12.165/27, 175.25.37.110/28, 163.128.170.0/19, 182.145.14.143/21. Tool sẽ tính ra cả Network, Broadcast, dải khả dụng và visualize bit để bạn dễ hình dung.

📐 Công thức áp dụng
  • Số bit Host = 32 − prefix
  • Số Host khả dụng = 2^(bit host) − 2
  • Network ID: tất cả bit Host = 0
  • Broadcast: tất cả bit Host = 1
